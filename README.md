# قالب وردپرس «خاص» (KHAS)

قالب مجله سبک‌زندگی راست‌چین (RTL) با پالت تیره، فیروزه‌ای و طلایی — بازسازی کامل طراحی خاص برای وردپرس.

---

## 📦 Repository

This is the **`khas-theme`** repository — an independent, standalone WordPress theme with **no dependency on Node.js, npm, or the companion `khas-pwa` Next.js application**. It can be installed on any standard WordPress hosting environment.

- 🚀 [`INSTALLATION.md`](./INSTALLATION.md) — local setup and first-run guide
- 🌐 [`DEPLOYMENT.md`](./DEPLOYMENT.md) — production deployment guide
- 📋 [`DEPENDENCIES.md`](./DEPENDENCIES.md) — full dependency / requirements list

## ✨ امکانات
- طراحی راست‌چین کامل با فونت **وزیرمتن**
- صفحه اصلی سفارشی: هیرو، دایره‌های دسته‌بندی، مقالات ویژه، آخرین مقالات، راهنماهای محبوب، خبرنامه، اینستاگرام
- خبرنامه با **AJAX** و ذخیره در جدول اختصاصی (`wp_khas_subscribers`) + صفحه مدیریت مشترکین
- صفحه تک‌نوشته با هیرو تمام‌عرض، مقالات مرتبط و دیدگاه‌ها
- آرشیو دسته‌بندی با آیکون ایموجی (قابل تنظیم برای هر دسته)
- زمان مطالعه خودکار و اعداد فارسی
- کاملاً ریسپانسیو + پشتیبانی از منو، لوگوی سفارشی، ویجت و ترجمه

## 🚀 نصب
۱. پوشه `khas-theme` را به صورت zip فشرده کنید (نام دلخواه، مثلاً `khas.zip`).
۲. در پیشخوان وردپرس: **نمایش › پوسته‌ها › افزودن پوسته › بارگذاری پوسته** و فایل zip را آپلود کنید.
۳. پوسته را **فعال** کنید (با فعال‌سازی، جدول مشترکین خبرنامه ساخته می‌شود).

## ⚙️ راه‌اندازی محتوا (برای شبیه شدن دقیق به دمو)
1. **صفحه اصلی:** یک برگه بسازید و در **تنظیمات › خواندن**، «نمایش صفحه اصلی» را روی آن برگه بگذارید. قالب به‌صورت خودکار از `front-page.php` استفاده می‌کند.
2. **مقالات ویژه:** پست‌هایی که می‌خواهید در بخش هیرو و «مقالات ویژه» دیده شوند را **سنجاق (Sticky)** کنید (ویرایش سریع › سنجاق کردن).
3. **تصاویر شاخص:** برای هر پست حتماً «تصویر شاخص» ست کنید تا در کارت‌ها و اینستاگرام نمایش داده شود.
4. **دسته‌بندی‌ها:** در ویرایش هر دسته، فیلد «آیکون (ایموجی)» را پر کنید (مثلاً 👗 یا ✈️). در غیر این صورت آیکون پیش‌فرض بر اساس نامک دسته انتخاب می‌شود.
5. **منو:** در **نمایش › فهرست‌ها** یک منو بسازید و به محل «منوی اصلی» اختصاص دهید (اختیاری — در نبود منو، دسته‌های پرکاربرد نمایش داده می‌شوند).

## 📬 خبرنامه
- ایمیل‌های ثبت‌شده در جدول `wp_khas_subscribers` ذخیره می‌شوند.
- لیست مشترکین از منوی **«خبرنامه خاص»** در پیشخوان قابل مشاهده است.

## 🗂 ساختار فایل‌ها
```
khas-theme/
├─ style.css              (هدر قالب + تمام استایل‌ها)
├─ functions.php          (راه‌اندازی، فونت، خبرنامه، هلپرها)
├─ header.php / footer.php
├─ front-page.php         (صفحه اصلی)
├─ index.php              (بلاگ / نتایج جستجو)
├─ single.php             (تک‌نوشته)
├─ page.php               (برگه)
├─ category.php / archive.php
├─ 404.php
├─ comments.php / searchform.php
├─ assets/main.js         (منوی موبایل + AJAX خبرنامه)
└─ template-parts/
   ├─ newsletter.php
   └─ pagination.php
```
